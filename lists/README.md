## ğŸ’¡ Key Concepts

- **Mutable**: Lists can be modified after creation
- **Ordered**: Elements maintain their insertion order
- **Dynamic**: Lists can grow and shrink as needed
- **Heterogeneous**: Can store different data types

## ğŸ”¥ Common Patterns

1. **Two Pointers**: Used for sorted arrays, palindromes, etc.
2. **Sliding Window**: For subarray problems with size k
3. **Prefix Sum**: For range sum queries
4. **Kadane's Algorithm**: Maximum subarray sum
5. **Dutch National Flag**: Three-way partitioning
